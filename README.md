# GlowUP – E-Commerce Website

GlowUP is an e-commerce website developed as part of a Software Engineering project. The system enables users to browse products, manage shopping carts, and complete purchases through a user-friendly interface.

---

##  Overview
GlowUP is designed as an online platform for trendy and quality apparel, serving customers, suppliers, and administrators. The system focuses on providing a scalable and structured solution for managing products, users, and transactions.

---

## Features
- Product browsing and filtering  
- Shopping cart management  
- Checkout and payment process  
- User account management  
- Product catalog management for admins and suppliers  

---

## Software Engineering Aspects
This project applies key Software Engineering concepts including:

- Requirements analysis  
- System design and modeling  
- Use case development  
- UML diagrams (Use Case, Class, ER)  
- UI/UX design  
- Structured system architecture  

---

## System Design
The system was designed using multiple Software Engineering artifacts:

- **Use Case Diagrams** – Define system interactions  
- **Class Diagram** – Represents system structure  
- **ER Diagram** – Models database relationships  
- **Context Diagram** – Shows system boundaries  
- **Traceability Matrix** – Links requirements to implementation  

---

## User Roles
- **Customer:** Browse products, manage cart, and complete purchases  
- **Admin:** Manage product catalog, inventory, and pricing  
- **Supplier:** Add and update products  

---

##  UI/UX Design
The system includes user-friendly interfaces such as:
- Home page with navigation and product categories  
- Product catalog with filtering and sorting  
- Shopping cart interface  
- Checkout and payment screens  

---

## Design Patterns
- **MVC (Model-View-Controller):** Separates logic, UI, and control  
- **Facade Pattern:** Simplifies complex operations like checkout
